# Joe home assistant
Joe is a home assistant.  

Joe is built to be highly modular and modable.

# Setup
Install python 3 then follow intructions for your platform.
## Windows
```
pip install SpeechRecognition
pip install pyttsx3
pip install pipwin
pipwin install PyAudio
```
## Linux
```bash
# nothing at all
```
